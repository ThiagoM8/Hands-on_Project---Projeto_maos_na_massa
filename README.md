# 🚀 Hands-on Project: Python Fundamentals 🇬🇧 🇧🇷

## 📄 Description / Descrição
This project was developed as part of the **Alura** "Mão na Massa" course. It focuses on Python logic, covering lists, dictionaries, and functions to manage a restaurant system. 🐍
Este projeto foi desenvolvido como parte do curso "Mão na Massa" da **Alura**. Ele foca em lógica Python, abrangendo listas, dicionários e funções para gerenciar um sistema de restaurantes. 🍴

## 💻 Tech Stack / Tecnologias
* **Language:** Python 3.12 🐍
* **Version Control:** Git & GitHub 📂
* **Main Concepts:** Sets, Dictionaries, and List Comprehension 🧠

## 🛠️ Key Features / Funcionalidades
* **✅ Restaurant Registration:** Add new establishments to the system.
* **🔄 Status Management:** Toggle restaurants between active and inactive.
* **⭐ Rating System:** Allow users to rate restaurants using a scale.

## 🚀 How to run / Como executar
1. **Clone this repository / Clone este repositório:**
   `git clone git@github.com:ThiagoM8/Hands-on-Project---Projeto-m-os-na-massa.git`
2. **Run the main file / Execute o arquivo principal:**
   `python app.py`
